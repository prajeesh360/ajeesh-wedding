# Ajeesh &amp; Archana — Wedding Invitation

A single-page wedding invitation site: countdown, families, event details with map, photo gallery, and e-invitation.

## Hosting with GitHub Pages

1. Go to **Settings &rarr; Pages** in this repo.
2. Under "Build and deployment", set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`.
3. Save — the site will be published at `https://prajeesh360.github.io/ajeesh-wedding/`.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server (e.g. `python -m http.server`).
